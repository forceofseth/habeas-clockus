Erstinstallation

Die App ist ad-hoc-signiert, aber nicht Apple-notarisiert. macOS sperrt eine frisch heruntergeladene Kopie daher zunächst.

  1. DMG öffnen und die App nach /Programme ziehen.
  2. Falls macOS „beschädigt“ meldet oder den Start blockiert, einmalig im Terminal ausführen: 
    xattr -dr com.apple.quarantine "/Applications/Habeas Clockus.app"
  3. Alternativ: Systemeinstellungen → Datenschutz & Sicherheit → „Trotzdem öffnen“.
Quellcode auf GitHub. Updates installiert die App nach der Erstinstallation automatisch.
